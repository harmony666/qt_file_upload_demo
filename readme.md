# 环境需求

python版本：3.9

pyqt版本：pyqt6

安装命令

`pip install -i https://pypi.tuna.tsinghua.edu.cn/simple pyqt6`

`pip install -i https://pypi.tuna.tsinghua.edu.cn/simple PySide6`
